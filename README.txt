HƯỚNG DẪN CÀI ĐẶT WEB
B1 : Bật Apache và MySQl trong phần mềm XAMPP
B2 : Truy cập phpMyAdmin và chạy câu lệnh sau để tạo database : `CREATE DATABASE digital_wallet`
B3 : Load lại trang và chọn database `digital_wallet`
B4 : Chọn `Nhập` hoặc `Import` tại thanh Menu phía trên
B5 : Tại phần tải lên tập tin : Chọn file `database.sql` trong Folder của project
B6 : Đưa tất cả nội dung bên trong Folder vào Folder `htdocs`
B7 : Mở File `config.php` và chỉnh sửa thông tin như tài khoản truy cập phpMyAdmin, tài khoản Email
B8 : Truy cập vào liên kết sau và cài đặt Email theo hướng dẫn phần 4 :
https://freetuts.net/huong-dan-gui-mail-trong-php-voi-phpmailer-710.html#goto-h2-3
B9 : Truy cập liên kết http://localhost/ để vào trang WEB
